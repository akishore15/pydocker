pyDocker

Installation

To install, you need to clone the repository on GitHub by running


        sudo apt install git && git clone https://github.com/akishore15/pydocker.git

if you did not already have Git installed, and if you did, run the command:


        git clone https://github.com/akishore15/pydocker.git
